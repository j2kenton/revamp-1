# Assignment: AI-Powered Chat Application

Design and implement UI for a Chat application, using React Typescript library. The user must be able to chat with an AI LLM.

## Tasks

### Authentication System

- Implement authentication using MSAL

### User Interface

- Build a responsive UI in React/NextJS using tailwind
- Implement UI framework with theming (AntD / Shadcn UI / Material UI)
- Implement dark/light mode
- Implement navigation using layout (TanStack Router / React Router v7)
- Implement TanStack query for API data fetching and async state management
- Implement request validation using validation library such as zod
- In case complex state management is required, using dedicated state management library such as zustand is recommended

## Deliverables

- Complete source code with proper comments
- Project walkthrough documentation
- Short video demonstration
